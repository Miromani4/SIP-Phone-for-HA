## SIP Doorbell v1.1.10 - Отдельный файл логирования

### Изменения

#### Исправлено логирование
- **Добавлена запись логов в отдельный файл**
  - Логи теперь пишутся в: `~/.homeassistant/sip_logs/sip_doorbell.log`
  - Ротация файлов: 10 файлов по 5MB каждый
  - Детальное форматирование с временными метками
  
#### Улучшения форматирования
- Формат лога: `YYYY-MM-DD HH:MM:SS - LEVEL - [funcName:line] - message`
- UTF-8 кодировка для корректного отображения кириллицы

### Файлы логов

Логи доступны по пути:
```bash
~/.homeassistant/sip_logs/
├── sip_doorbell.log          # Текущий лог
├── sip_doorbell.log.1        # Ротированный лог
├── sip_doorbell.log.2
└── ...
```

### Диагностика

Теперь в логах можно найти:
- Старт SIP-соединения с параметрами
- Попытки регистрации с номером
- Потерю соединения с счетчиком
- Таймауты с полным контекстом
- Все ошибки и предупреждения

### Для просмотра логов в реальном времени

```bash
tail -f ~/.homeassistant/sip_logs/sip_doorbell.log
```

### Установка

Обновите интеграцию через HACS или вручную.

---

**Совместимость:** Home Assistant 2023.1+
