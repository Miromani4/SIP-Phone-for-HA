# Release v1.1.10 - File Logging for Diagnostics

## 📝 Changes

- Добавлено логирование в отдельный файл (`~/.homeassistant/sip_logs/sip_doorbell.log`)
- Логи пишутся одновременно в Home Assistant и в отдельный файл
- Файл логов находится в `~/.homeassistant/sip_logs/`
- Формат логов: timestamp, уровень логирования, сообщение
- Полезно для диагностики без просмотра всего журнала HA

## 📁 Files Changed

- `custom_components/sip_doorbell/__init__.py` - добавлен отдельный FileHandler

## ⚠️ Breaking Changes

Нет

## 🔧 Configuration

Директория для логов создаётся автоматически при инициализации интеграции.

## 🐛 Known Issues

- Убедитесь что у Home Assistant есть права на запись в `~/.homeassistant/sip_logs/`
