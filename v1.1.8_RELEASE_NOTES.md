# SIP Doorbell v1.1.8

## 🐛 Исправления ошибок
- Исправлена синтаксическая ошибка в `call_popup.js` (scrimClickAction)
- Добавлены проверки безопасности в `phone.py` перед вызовом `sendto()` для предотвращения ошибок при отсутствии `remote_addr`

## ✨ Новые возможности
### Расширенная кастомизация всплывающего окна вызова
- **Позиция окна**: `center`, `top`, `bottom-right`, `bottom-left`, `top-right`, `top-left`
- **Тема оформления**: `light`, `dark`, `auto` (автоопределение темы Home Assistant)
- **Поддержка камеры**: отображение видео с указанной камеры в карточке вызова
- **Звуковые уведомления**: `play_sound`, `sound_url` для настройки звука звонка
- **Вибрация** на мобильных устройствах (`vibrate: true`)
- **Кастомные кнопки действий**: `custom_actions` с поддержкой:
  - Вызова сервисов Home Assistant
  - Отправки DTMF сигналов
  - Настраиваемых иконок и текста
- **Цвета кнопок**: настраиваемые `colors.answer`, `colors.decline`, `colors.dtmf`
- **Размеры окна**: `width`, `height`
- **Автозакрытие**: `auto_close`, `timeout` для автоматического закрытия окна

## 📝 Пример конфигурации

```yaml
- type: custom:sip-dorbell-popup
  extension: "100"
  position: "bottom-right"
  theme: "auto"
  camera_entity: "camera.doorbell"
  play_sound: true
  sound_url: "/local/sounds/ringtone.mp3"
  vibrate: true
  width: "400px"
  height: "auto"
  auto_close: false
  timeout: 60
  colors:
    answer: "#4CAF50"
    decline: "#F44336"
    dtmf: "#2196F3"
  custom_actions:
    - name: "Открыть дверь"
      icon: "mdi:door-open"
      service: "switch.turn_on"
      service_data:
        entity_id: "switch.door_relay"
    - name: "DTMF 1"
      icon: "mdi:numeric-1-circle"
      dtmf: "1"
```
