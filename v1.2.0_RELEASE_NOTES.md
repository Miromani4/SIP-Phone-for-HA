# SIP Doorbell v1.2.0 - Release Notes

**Release Date:** 2026-03-31

## 🚀 New Features

### 1. WebRTC Real-time Audio Support
- **Native WebRTC** для двустороннего аудио
- Эхо-подавление и шумоподавление
- Низкая задержка передачи звука
- Автоматический выбор кодеков

### 2. Media Player Platform
- Новая сущность `media_player` для каждого SIP-аккаунта
- Воспроизведение мелодий звонка
- Управление громкостью через Home Assistant
- Интеграция с автоматизациями

### 3. Всплывающее окно звонка с WebRTC
- Модальное окно при входящем звонке
- **Кнопка "Ответить"** с WebRTC-аудио
- Кнопка "Отклонить"
- Кастомизируемый интерфейс (см. ниже)

## 🔧 Technical Improvements

### WebSocket API
- `sip_doorbell/webrtc_offer` - обработка WebRTC предложений
- `sip_doorbell/webrtc_ice_candidate` - ICE-кандидаты
- `sip_doorbell/webrtc_close` - закрытие соединения

### JavaScript API
- `SIPWebRTCClient` класс для управления звонками
- Автоматическая подписка на события звонков
- Методы управления аудио потоком

## 📋 Customization Options

Всплывающее окно звонка можно кастомизировать через Home Assistant themes:

```yaml
# configuration.yaml
frontend:
  themes:
    my_sip_theme:
      # Цвета кнопок
      sip-popup-primary-color: "#4CAF50"
      sip-popup-decline-color: "#F44336"
      
      # Размеры окна
      sip-popup-width: "450px"
      sip-popup-height: "auto"
      
      # Прозрачность фона
      sip-popup-overlay-opacity: "0.7"
      
      # Шрифты
      sip-popup-font-family: "Roboto, sans-serif"
      sip-popup-title-size: "1.5em"
      
      # Анимации
      sip-popup-animation: "slideInRight"
      # Варианты: slideInRight, slideInLeft, fadeIn, bounceIn
```

### Дополнительные CSS переменные:
| Переменная | Описание | Значение по умолчанию |
|------------|----------|----------------------|
| `--sip-popup-border-radius` | Радиус скругления углов | `12px` |
| `--sip-popup-box-shadow` | Тень окна | `0 4px 20px rgba(0,0,0,0.3)` |
| `--sip-popup-bg-color` | Фон окна | `#1c1c1c` |
| `--sip-popup-text-color` | Цвет текста | `#ffffff` |
| `--sip-popup-mute-color` | Цвет кнопки mute | `#FFC107` |

## 🔄 Migration from v1.1.x

1. **Backup** your current configuration
2. Update to v1.2.0 via HACS or manually
3. **Restart** Home Assistant
4. New `media_player` entities will be auto-discovered
5. **Optional:** Add WebRTC JavaScript to your dashboard:

```yaml
# configuration.yaml
lovelace:
  resources:
    - url: /local/sip_doorbell/call-popup.js
      type: module
```

## 🐛 Known Issues

1. **Firefox:** может требовать дополнительных разрешений для микрофона
2. **iOS Safari:** WebRTC работает только в PWA-режиме
3. **Android Chrome:** рекомендуется использовать HTTPS

## 📝 Configuration Example

```yaml
# automation.yaml
- alias: "Show SIP popup on incoming call"
  trigger:
    - platform: event
      event_type: sip_doorbell_incoming_call
  action:
    - service: browser_mod/popup
      data:
        title: "Incoming Call"
        content: |
          <sip-call-popup
            caller="{{ trigger.event.data.caller_name }}"
            webrtc-enabled="true">
          </sip-call-popup>
```
