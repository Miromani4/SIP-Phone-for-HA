## 🔧 Fix Release v1.2.1

### Исправлено
- **Custom element doesn't exist**: Карточка теперь корректно регистрируется как Custom Card для Home Assistant Dashboard
- Добавлена регистрация через `window.customCards` для отображения в UI picker

### Использование
Теперь карточку можно добавить через UI:
```yaml
type: custom:sip-doorbell-call-popup
title: 🔔 Doorbell
auto_open: true
auto_close: true
timeout: 30
webrtc_enabled: true
```

### Установка
1. Обновите через HACS до v1.2.1
2. Очистите кэш браузера (Ctrl+F5)
3. Добавьте карточку на Dashboard
